# SKILL_VETTER - 技能审查规范

**强制规则：安装任何技能前，必须先运行 skill-vetter 扫描**

## 触发条件
- 从 ClawdHub/GitHub/其他来源安装技能
- 运行未知来源的技能
- 其他 agent 推荐的技能

## 审查流程（4步法）

### Step 1: 来源检查
- [ ] 作者是否知名/可信？
- [ ] 下载量/星标数？
- [ ] 最后更新时间？
- [ ] 其他 agent 的评价？

### Step 2: 代码审查（强制）
**🚨 发现以下任一内容，立即拒绝：**
- curl/wget 到未知 URL
- 发送数据到外部服务器
- 请求凭证/API key
- 读取 ~/.ssh, ~/.aws, ~/.config
- 访问 MEMORY.md, USER.md, SOUL.md, IDENTITY.md
- base64 解码操作
- eval() 或 exec() 执行外部输入
- 修改系统文件（workspace 外）
- 网络调用使用 IP 而非域名
- 混淆/压缩/编码的代码
- 请求 sudo/ elevated 权限
- 访问浏览器 cookie/session

### Step 3: 权限范围评估
- [ ] 需要读取哪些文件？
- [ ] 需要写入哪些文件？
- [ ] 运行哪些命令？
- [ ] 需要网络访问？到哪儿？
- [ ] 权限范围是否最小化？

### Step 4: 风险分级
| 级别 | 示例 | 操作 |
|------|------|------|
| 🟢 LOW | 笔记、天气、格式化 | 基础审查，可安装 |
| 🟡 MEDIUM | 文件操作、浏览器、API | 完整代码审查 |
| 🔴 HIGH | 凭证、交易、系统 | 需人工批准 |
| ⛔ EXTREME | 安全配置、root 权限 | 禁止安装 |

## 输出报告格式
```
【技能审查报告】
───────────────────────────────────────
技能: [name] | 风险等级: [🟢/🟡/🔴/⛔]
来源: [ClawdHub/GitHub/其他]
所需权限: [文件/网络/命令列表]
结论: [✅ 安全安装 / ⚠️ 谨慎安装 / ❌ 禁止安装]
───────────────────────────────────────
```

## 信任层级
1. **OpenClaw 官方技能** → 降低审查标准（仍需审查）
2. **高星仓库 (1000+)** → 中等审查
3. **已知作者** → 中等审查
4. **新/未知来源** → 最高审查
